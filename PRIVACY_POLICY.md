# Privacy Policy for CAMORENT Admin

**Last Updated: January 5, 2026**

## Introduction

CAMORENT ("we", "our", or "us") operates the CAMORENT Admin mobile application (the "App"). This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our administrative application for managing camera equipment rental bookings.

By using the CAMORENT Admin App, you agree to the collection and use of information in accordance with this policy.

## Information We Collect

### Personal Information
When you use the CAMORENT Admin App, we collect and process the following personal information:

**Administrator Account Information:**
- Phone number (used for login authentication)
- Password (encrypted and securely stored)
- Admin profile information

**Customer Booking Data:**
- Customer names and contact information
- Booking details (dates, times, equipment, locations)
- Rental duration and pricing information
- Booking status and approval history

**Payment Information:**
- Invoice details
- Payment transaction records
- Payment status and history

**Device Information:**
- Device push notification tokens
- Device type and operating system
- App version and usage data

### Automatically Collected Information
- App usage statistics
- Error logs and crash reports
- API request logs for troubleshooting

## How We Use Your Information

We use the collected information for the following purposes:

1. **Authentication & Access Control**
   - Verify your identity as an authorized administrator
   - Maintain secure access to the admin dashboard

2. **Booking Management**
   - Display pending, ongoing, and completed bookings
   - Enable approval/rejection of booking requests
   - Facilitate communication with customers

3. **Notifications**
   - Send push notifications for new booking requests
   - Alert you about booking status changes
   - Provide real-time updates on rental activities

4. **Invoice Management**
   - Generate and send invoices to customers
   - Track payment status
   - Send payment reminders

5. **Service Improvement**
   - Analyze app usage to improve functionality
   - Fix bugs and technical issues
   - Enhance user experience

## Data Storage and Security

### Security Measures
We implement industry-standard security measures to protect your information:

- **Encrypted Communication:** All data transmitted between the app and our servers uses HTTPS/TLS encryption
- **Secure Storage:** Authentication tokens are stored using device-level encryption
- **Access Controls:** Only authorized administrators can access the system
- **Token-Based Authentication:** JWT (JSON Web Tokens) for secure session management
- **No Data Backups on Device:** Android backup is disabled to prevent unauthorized data extraction

### Data Retention
- Active session data: Retained while you're logged in
- Booking records: Retained as per business requirements and legal obligations
- Deleted account data: Removed from active systems within 30 days of account deletion
