## Build & Submission Commands

### Install Dependencies
```bash
npm install
```

### Build AAB for Play Store
```bash
# Production build (Android App Bundle)
npx eas-cli build --platform android --profile production

# If you need APK for testing
npx eas-cli build --platform android --profile production-apk
```

### Submit to Play Store (After Build)
```bash
# Automated submission (requires service account JSON)
npx eas-cli submit --platform android --profile production
```

**OR**

- Download the `.aab` file from EAS build
- Manually upload to Google Play Console
- Fill in store listing information
- Submit for review
